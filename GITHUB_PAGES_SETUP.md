# Instructions pour configurer GitHub Pages manuellement

ÉTAPE 1: Aller sur la page des paramètres
=======================================
URL: https://github.com/neshop8/claw-code/settings/pages

ÉTAPE 2: Configurer GitHub Pages
================================
- Source: "Deploy from a branch"
- Branch: "main"
- Folder: "/" (racine)

ÉTAPE 3: Cliquer sur "Save"
==========================

ÉTAPE 4: Attendre 2-3 minutes
============================

ÉTAPE 5: Tester l'URL
=====================
https://neshop8.github.io/claw-code/

Si ça ne marche pas:
- Vider le cache du navigateur (Ctrl+F5)
- Attendre 5-10 minutes de plus
- Vérifier que le fichier index.html est bien à la racine de la branche main
